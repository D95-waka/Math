\[
	(2x, 2y, 2z) = \lambda (8x, 0, 30z - 40),
	\qquad
	0 = 4x^2 + 25 - 40z + 15z^2
\]
ולכן
\[
	2y = 0 \implies y = 0
\]
ומהשוויון הראשון נקבל
\[
	2x = \lambda 8 x
\]
אם $\lambda = \frac{1}{4}$ אז השוויון מתקיים לכל $x$ ונצטרך למצוא דרך אחרת לצמצם את התוצאות שלו.
\[
	2x = \lambda (30z - 40) = \frac{30}{4}z - 10
\]
